# Privacy Policy

AI Water Meter estimates the operational water, energy, and carbon impact of AI chat sessions.

## Data Processed

The extension reads visible text from supported AI chat pages so it can estimate prompt and response length. Processing happens locally in the browser.

## Data Stored

The extension stores daily aggregate estimates in browser extension storage. These records include estimated tokens, water, energy, and carbon totals. They do not include prompt text, response text, account identifiers, or personal contact information.

## Data Shared

The extension does not transmit chat content, estimates, browsing history, analytics, or personal information to any external server.

## Permissions

The extension requests `storage` so it can save local daily totals and preferences. It requests host access for supported AI chat sites so the in-page meter can read visible chat text on those pages.

## Remote Code

The extension does not load or execute remotely hosted code. All extension logic is packaged with the extension.
